## 我在做什么

- 用 DEM、遥感、栅格处理和空间建模理解城市洪涝、海岸蓝碳和土地利用变化。
- 把研究流程里的数据处理、实验记录、质量检查和复现步骤工程化。
- 持续打磨 AI 编码协作、仓库治理、Hooks、文档路由和安全边界。
